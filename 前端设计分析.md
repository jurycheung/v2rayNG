# v2rayNG 前端设计分析

**分析日期:** 2026-03-13  
**版本:** v2.0.14  
**技术栈:** Kotlin + Material Design 3

---

## 📋 概述

v2rayNG 采用现代化的 Android 前端架构，基于 **Material Design 3** 设计规范，使用 **Kotlin** 作为主要开发语言，遵循 **MVVM 架构模式**。

## 🎨 UI 设计

### 设计系统：Material Design 3

**主题配置:** `themes.xml`
```xml
<style name="AppThemeBase" parent="Theme.Material3.DayNight">
    <!-- 主色调：黑色 -->
    <item name="colorPrimary">@color/md_theme_primary</item>
    
    <!-- 强调色：橙色 -->
    <item name="colorSecondary">@color/md_theme_secondary</item>
    
    <!-- 辅助色：绿色 -->
    <item name="colorTertiary">@color/md_theme_tertiary</item>
</style>
```

### 配色方案

| 颜色类型 | 颜色值 | 用途 |
|---------|--------|------|
| **Primary** | `#000000` (黑) | 主色调、文字 |
| **Secondary** | `#f97910` (橙) | 强调色、FAB 按钮 |
| **Tertiary** | `#009966` (绿) | 成功状态、连接指示 |
| **Error** | `#BA1A1A` (红) | 错误提示 |
| **Surface** | `#FFFFFF` (白) | 背景、卡片 |

### 支持深色模式

- 使用 `Theme.Material3.DayNight` 主题
- 自动根据系统设置切换
- 颜色资源在 `colors.xml` 中定义

## 📱 核心界面

### 1️⃣ 主界面 (MainActivity)

**布局文件:** `activity_main.xml`

**结构:**
```
DrawerLayout
├── AppBarLayout
│   └── Toolbar (顶部导航栏)
├── CoordinatorLayout
│   ├── LinearLayout
│   │   ├── ProgressBar (加载进度)
│   │   ├── TabLayout (分组切换)
│   │   ├── ViewPager2 (服务器列表)
│   │   └── TestView (连接测试)
│   └── FloatingActionButton (启动/停止按钮)
└── NavigationView (侧边抽屉菜单)
```

**功能:**
- ✅ 服务器列表展示（支持分组）
- ✅ 一键启动/停止服务
- ✅ 连接状态测试
- ✅ 多种导入方式（二维码、剪贴板、文件）
- ✅ 侧边菜单导航

**特色设计:**
- **双 FAB 按钮:** 主 FAB(启动/停止) + 定位 FAB(快速定位)
- **Tab 分组:** 支持多订阅分组切换
- **底部测试栏:** 显示连接状态和延迟

## 🔧 技术实现

### ViewBinding

所有 Activity 使用 **ViewBinding** 进行视图绑定：
```kotlin
private val binding by lazy {
    ActivityMainBinding.inflate(layoutInflater)
}
```

**优势:**
- 类型安全
- 空指针安全
- 编译时检查

### ViewModel + LiveData

使用 Android Architecture Components：
```kotlin
class MainActivity : HelperBaseActivity() {
    val mainViewModel: MainViewModel by viewModels()
    
    override fun onCreate(savedInstanceState: Bundle?) {
        mainViewModel.isRunning.observe(this) { isRunning ->
            applyRunningState(false, isRunning)
        }
    }
}
```

### RecyclerView + Adapter

列表展示使用 RecyclerView：
- `MainRecyclerAdapter` - 服务器列表
- `SubSettingRecyclerAdapter` - 订阅列表
- `PerAppProxyAdapter` - 应用列表
- `UserAssetAdapter` - 资源文件列表

### 协程 (Kotlin Coroutines)

异步操作使用协程：
```kotlin
lifecycleScope.launch(Dispatchers.IO) {
    val result = mainViewModel.updateConfigViaSubAll()
    withContext(Dispatchers.Main) {
        // 更新 UI
    }
}
```

## 🔄 数据流

### 配置导入流程

```
用户操作 (扫码/剪贴板/文件)
    ↓
MainActivity.importBatchConfig()
    ↓
AngConfigManager.importBatchConfig()
    ↓
解析配置 → 验证 → 存储
    ↓
MainViewModel.reloadServerList()
    ↓
RecyclerView 刷新
```

### 服务启动流程

```
点击 FAB 按钮
    ↓
检查 VPN 权限
    ↓
V2RayServiceManager.startVService()
    ↓
启动后台服务
    ↓
更新 UI 状态 (FAB 图标变化)
```
